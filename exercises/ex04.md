# Ex04 — Registrar primeira tool MCP

## Contexto

Com o servidor criado, registre a primeira capacidade MCP: `fetch_ticket_context`.

## Arquivo permitido

```text
stack_sentinel/mcp_server/server.py
```

## Contrato

Implemente:

```python
register_ticket_tool(server: SimpleMCPServer) -> SimpleMCPServer
```

Regras:

- registre uma `ToolDefinition`;
- use `name == TICKET_TOOL_NAME`;
- use `handler == tools.fetch_ticket_context`;
- inclua descrição não vazia;
- inclua schema com `ticket_id` como argumento obrigatório;
- retorne o mesmo servidor recebido, agora modificado.

## Validação

```bash
python run.py test ex04
```

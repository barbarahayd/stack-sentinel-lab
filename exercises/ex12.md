# Ex12 — Node que consulta ticket

## Contexto

Quando a rota for ticket, o agente deve consultar uma tool MCP e salvar o resultado no state.

## Arquivo permitido

```text
stack_sentinel/agent/nodes.py
```

## Contrato

Implemente:

```python
fetch_ticket_node(state: AgentState, mcp_client: MCPClient) -> AgentState
```

Regras:

- use `state["ticket_id"]` se ele já existir;
- se não existir, extraia o ID de `state["user_input"]` usando o padrão `TCK-000`;
- chame `mcp_client.call_tool(TICKET_TOOL_NAME, {"ticket_id": ticket_id})`;
- quando `ok=True`, salve o resultado em `context`, preencha `ticket_id` e deixe `error` como `None`;
- quando não houver ID ou a tool falhar, preencha `error` e não quebre a execução.

## Validação

```bash
python run.py test ex12
```

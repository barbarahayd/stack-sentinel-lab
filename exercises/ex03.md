# Ex03 — Criar MCP server mínimo

## Contexto

Antes de registrar qualquer capacidade, o servidor MCP do domínio Stack Sentinel precisa existir com identidade e metadados.

## Arquivo permitido

```text
stack_sentinel/mcp_server/server.py
```

## Contrato

Implemente:

```python
create_mcp_server() -> SimpleMCPServer
```

O servidor deve ter:

- `name == "stack-sentinel-mcp"`;
- `description` não vazia;
- `metadata["domain"] == "incident-investigation"`;
- `metadata["version"] == "0.1.0"`;
- coleções de tools, resources e prompts inicialmente vazias.

## Validação

```bash
python run.py test ex03
```

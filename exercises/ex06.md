# Ex06 — Criar resource MCP

## Contexto

Nem tudo é ação. Algumas informações são documentação, política ou contexto. Resources representam dados legíveis.

## Arquivo permitido

```text
stack_sentinel/mcp_server/resources.py
```

## Contrato

Implemente:

```python
read_doc_resource(uri: str, client: Optional[MockServiceClient] = None) -> dict
```

Em sucesso, retorne:

```python
{
    "ok": True,
    "uri": "docs://incident-response",
    "title": "Incident Response Guide",
    "content": "..."
}
```

Em falha, retorne `ok=False` e `error`.

Regras:

- aceite apenas URIs presentes em `RESOURCE_TO_SLUG`;
- use `client.get_doc(slug)`;
- preserve o campo `uri` original na resposta.

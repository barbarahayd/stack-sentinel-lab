# Ex07 — Criar prompt MCP

## Contexto

Prompts MCP padronizam como clientes e agentes devem agir dentro de um domínio. Aqui, o prompt orienta uma triagem de incidente.

## Arquivo permitido

```text
stack_sentinel/mcp_server/prompts.py
```

## Contrato

Implemente:

```python
incident_triage_prompt(user_question: str, available_context: str) -> str
```

O texto retornado deve orientar o agente a:

- resumir o problema;
- citar severidade quando houver evidência;
- sugerir próximo passo;
- não inventar dados ausentes;
- usar a pergunta do usuário;
- usar o contexto disponível.

## Validação

```bash
python run.py test ex07
```

# Ex08 — Criar grafo mínimo

## Contexto

Antes de integrar MCP, crie um grafo mínimo para entender a mecânica de execução: um state entra, passa por um node e sai atualizado.

## Arquivo permitido

```text
stack_sentinel/agent/graph.py
```

## Contrato

Implemente:

```python
compile_minimal_graph() -> SimpleGraph
```

Regras:

- retorne uma instância de `SimpleGraph`;
- adicione pelo menos um node usando `graph.add_node`;
- ao executar `graph.run({"user_input": "ping"})`, o resultado deve conter `final_answer == "ping"`;
- o node deve preservar `user_input`.

## Validação

```bash
python run.py test ex08
```

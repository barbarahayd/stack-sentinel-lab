# MCP ou tools diretas?

Use tool direta no agente quando:

- o protótipo é pequeno;
- a capacidade só existe para aquele agente;
- você quer reduzir camadas.

Use MCP quando:

- a capacidade representa um serviço ou domínio reutilizável;
- múltiplos clientes podem consumir a mesma capacidade;
- você quer padronizar tools, resources e prompts;
- faz sentido separar agente de adaptador de serviço.
